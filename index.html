Os dados são do Portal de Dados Abertos do Tribunal Superior Eleitoral. O script em R para tratar os dados e o código do site estão no meu GitHub. O cálculo do percentual dos votos válidos considera a quantidade de votos recebidas por todos os candidatos. Por isso, no primeiro turno, a soma dos dois candidatos exibidos não dá 100%. Este projeto optou por exibir no primeiro turno apenas os dois candidatos que disputaram o segundo turno das eleições. A plataforma desenvolvida por Gabriela Caesar em junho de 2022 durante o Master no Insper, com ajuda de Tiago Maranhão e Vinicius Sueiro.
